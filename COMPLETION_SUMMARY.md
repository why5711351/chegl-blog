## 🛠️ 网站维护

### 添加新文章

```bash
cd /root/.openclaw/workspace/adsense-blog

# 创建新文章
hugo new posts/your-article-title.md

# 编辑文章内容
# 设置 draft: false

# 本地测试
hugo server --buildFuture

# 提交并推送
git add .
git commit -m "Add new post: [标题]"
git push
```

Netlify 会自动重新部署（约 1-2 分钟）。

### 查看部署日志

访问：https://app.netlify.com → 选择站点 → Deploys

### 更新配置

编辑 `hugo.toml`，然后：
```bash
git add .
git commit -m "Update config"
git push
```
