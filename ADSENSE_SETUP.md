## 📋 AdSense 代码集成步骤

### 1. 创建 AdSense 账号

1. 访问 https://www.google.com/adsense
2. 使用 Google 账号登录
3. 填写网站信息（chegl.com）
4. 填写付款信息
5. 提交审核申请

### 2. 获取 AdSense 代码

审核通过后，你会获得：
- **Publisher ID**: `ca-pub-XXXXXXXXXXXXXXXX`
- **Ad Slot ID**: `XXXXXXXXXX`

### 3. 替换代码中的占位符

**文件 1**: `themes/lawyer/layouts/partials/ad-unit.html`
```html
<!-- 第 16 行 -->
data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"  ← 替换为你的 Publisher ID

<!-- 第 17 行 -->
data-ad-slot="XXXXXXXXXX"  ← 替换为你的 Ad Slot ID
```

**文件 2**: `themes/lawyer/layouts/partials/cookie-banner.html`
```javascript
// 第 103 行
script.src = 'https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX';
// 替换为你的 Publisher ID

// 第 115 行
gtag('config', 'G-XXXXXXXXXX');
// 替换为你的 GA4 Measurement ID
```

### 4. 测试广告显示

1. 清除浏览器缓存
2. 访问网站
3. 接受 Cookie
4. 检查广告是否显示

**注意**: 新 AdSense 账号可能需要 24-48 小时才能开始显示广告。

## 💡 AdSense 优化技巧

### 1. 广告密度控制

**推荐**:
- 每 1000 字：2-3 个广告
- 每页面：3-6 个广告
- 广告占比：<20% 页面面积

**避免**:
- 广告过多（影响体验）
- 广告遮挡内容
- 误导性广告标签

### 2. 广告位置优化

**高收益位置**:
- 内容顶部（首屏）
- 内容中部（阅读流程中）
- 侧边栏（桌面端）
- 内容底部（相关文章前）

**低收益位置**:
- 页脚（容易被忽略）
- 侧边栏底部（滚动不到）
- 弹出窗口（影响体验）

### 3. 广告格式选择

**推荐格式**:
- 展示广告（300x250, 336x280, 728x90）
- 信息流广告（自适应）
- 链接单元（辅助）

**避免格式**:
- 弹窗广告
- 自动播放视频
- 大型插页式

### 4. 移动端优化

```css
/* 移动端广告尺寸 */
@media (max-width: 768px) {
    .ad-container {
        margin: 1rem 0;
    }
    
    /* 使用响应式广告 */
    ins.adsbygoogle[data-ad-format="auto"] {
        width: 100%;
    }
}
```

### AdSense 报告指标

**关键指标**:
- 展示次数
- 点击次数
- 点击率 (CTR)
- 每次点击费用 (CPC)
- 每千次展示收益 (RPM)
- 总收入

**目标基准**（法律类）:
- CTR: 1-2%
- CPC: $50-150
- RPM: $30-80

### 收益优化策略

1. **提高流量**
   - SEO 优化
   - 内容营销
   - 社交媒体

2. **提高 CTR**
   - 广告位置测试
   - 广告格式优化
   - 内容与广告相关性

3. **提高 CPC**
   - targeting 高价值关键词
   - 创建高 CPC 主题内容
   - targeting 美国用户

## ⚠️ 常见问题

### Q: AdSense 审核多久？
A: 通常 2 天 -2 周，法律类可能更长。

### Q: 审核被拒怎么办？
A: 
1. 查看拒绝原因邮件
2. 解决所有问题
3. 添加新内容（10+ 篇）
4. 等待 1 周后重新申请

### Q: 如何避免无效点击？
A:
- 不要点击自己的广告
- 不要鼓励他人点击
- 监控异常点击模式
- 使用无效点击保护

### Q: 收益如何发放？
A:
- 达到$100 阈值后发放
- 每月 21-26 日
- 银行转账或支票
- 需要填写税务信息

### Q: 法律类 AdSense 有限制吗？
A:
- 不能保证案件结果
- 不能提供免费法律建议
- 需要明确披露是广告
- 需要资质声明
