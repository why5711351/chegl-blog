# 🎯 AdSense 盈利中心策略 - FindLegalHelp

## ⚠️ 法律利基 AdSense 特殊性

### YMYL (Your Money Your Life) 要求

Google 对法律、医疗、金融类网站审核极严，因为内容可能影响用户的：
- 财务稳定
- 健康安全
- 法律权益
- 人生重大决策

### AdSense 法律类 CPC 优势

**优势**: 法律类 CPC 最高 ($50-500+/点击)  
**挑战**: 审核严格、需要专业资质展示  
**机会**: 一旦通过，收益远高于普通博客

### 3. AdSense 代码集成

#### 自动广告 vs 手动广告

**推荐**: 手动广告（更好控制位置和密度）

**原因**:
- 法律网站需要保持专业性
- 避免广告出现在不合适位置
- 优先展示转化表单（lead 价值 > AdSense）

#### 广告密度控制

```
✅ 推荐: 每 1000 字 2-3 个广告
❌ 避免: 每段都有广告（影响体验）

页面广告总数:
- 首页：4-6 个
- 文章页：3-5 个
- 实践领域页：4-5 个
```

## 💡 AdSense 优化技巧

### 1. 广告格式选择

**推荐格式**（法律类）:

| 格式 | 尺寸 | 位置 | 预期收益 |
|------|------|------|---------|
| 展示广告 | 728x90 | 顶部、底部 | 高 |
| 展示广告 | 300x250 | 侧边栏、内容中 | 最高 |
| 展示广告 | 336x280 | 内容中部 | 高 |
| 链接单元 | 180x90 | 侧边栏 | 中 |
| 信息流广告 | 自适应 | 文章列表 | 中高 |

**避免格式**:
- ❌ 弹窗广告（影响体验）
- ❌ 自动播放视频（不专业）
- ❌ 大型插页式（阻碍内容）

### 2. 广告颜色匹配

```css
/* 广告样式与网站融合 */
.ad-container {
  border: 1px solid #e5e7eb;
  border-radius: 0.5rem;
  padding: 1rem;
  background: #f9fafb;
  margin: 2rem 0;
}

/* 标注为广告（合规要求） */
.ad-label {
  font-size: 0.75rem;
  color: #6b7280;
  margin-bottom: 0.5rem;
  text-transform: uppercase;
}
```

### 3. 移动端广告优化

```
移动端广告位:
- 顶部横幅（320x50）
- 内容中部（300x250）
- 底部固定（320x50）

注意事项:
✅ 广告不遮挡内容
✅ 点击区域足够大
✅ 加载速度快
❌ 避免过多广告（影响体验）
```

### 4. 广告与转化平衡

**优先级原则**:

```
1. 转化表单（lead 价值 $100+）
2. 电话咨询追踪（lead 价值 $200+）
3. AdSense 广告（点击价值 $50-150）
4. 其他变现（affiliate 等）
```

**实施策略**:
- 侧边栏：表单优先，广告其次
- 文章末尾：相关文章 + 广告
- 首页：实践领域 + 广告

## ⚠️ AdSense 政策合规

### 禁止内容（会导致拒绝）

- ❌ 保证案件结果
- ❌ 虚假资质声称
- ❌ 未经审核的法律建议
- ❌ 敏感犯罪细节描述
- ❌ 暴力、血腥内容
- ❌ 成人相关内容
- ❌ 仇恨言论、歧视
- ❌ 盗版、侵权内容

### 必需披露

- ✅ "这不是法律建议" 声明
- ✅ 律师广告标识
- ✅ 结果不保证声明
- ✅ 利益冲突披露
- ✅ 付费推荐披露

### 隐私合规

- ✅ Cookie 同意横幅
- ✅ 隐私政策链接
- ✅ 数据使用说明
- ✅ 第三方广告披露
- ✅ GDPR/CCPA 合规

## 🆘 AdSense 被拒应对

### 常见拒绝原因

1. **"Low Value Content"**
   - 解决：增加原创深度内容
   - 目标：每篇 2000+ 字

2. **"Site Not Ready"**
   - 解决：完善网站结构
   - 目标：50+ 篇内容

3. **"Policy Violation"**
   - 解决：检查所有合规声明
   - 目标：移除敏感内容

4. **"Insufficient Content"**
   - 解决：持续发布内容
   - 目标：3 个月运营历史

### 重新申请策略

- 等待至少 1 周
- 解决所有拒绝原因
- 添加新内容（10+ 篇）
- 改进网站设计
- 重新提交申请
